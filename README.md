## JavaCalculator
This is a simple Calculator made with Java. The idea behind this project was to learn how to use JFrame. 

# Technologies

- Java
- Maven
- java.awt

# Features:

- Basics operations for integers and decimal numbers
- Shows graphical calculator with buttons to press.
- Has a Clear and a Delete button
- Shows the last operation made (little change history)
